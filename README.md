# AI Exam Checker

AI-powered exam answer evaluation system using OCR and BERT.

## Features
- OCR text extraction using Tesseract
- AI answer evaluation using BERT
- Student and Teacher dashboard
- Flask backend
- Bootstrap UI

## Technologies Used
- Python
- Flask
- Tesseract OCR
- Transformers (BERT)
- HTML, CSS, Bootstrap
- SQLite

## How to Run

```bash
pip install -r requirements.txt
python app.py
